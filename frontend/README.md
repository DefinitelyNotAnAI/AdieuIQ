# Contoso Resorts - Guest Experience Intelligence Platform

React + TypeScript frontend for guest recommendation and experience management system.

## Requirements

- Node.js 18+
- npm or yarn

## Installation

```bash
npm install
```

## Running Locally

```bash
# Development server (with hot reload)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Testing

```bash
# Run unit tests
npm test

# Run tests in watch mode
npm run test:watch

# Run with coverage
npm run test:coverage

# Run E2E tests
npm run test:e2e
```

## Code Quality

```bash
# Lint code
npm run lint

# Format code
npm run format
```

## Project Structure

```
frontend/
├── src/
│   ├── components/      # React components
│   ├── pages/           # Page components
│   ├── services/        # API client and auth
│   └── App.tsx          # Root component
└── tests/               # Test suites
```
